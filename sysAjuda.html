Manual do Sr. Especialista

Bem vindo ao Sr. Especialista, um software desenvolvido para guardar e organizar dados e informações de forma a disponibilizar conhecimento de qualquer especialidade.

Este manual ajudará a você entender o funcionamento do fluxo de trabalho. Todas as partes do programa com uma funcionalidade específica foram separadas em "cartões" e cada seção possui cartões relacionados para cumprir um objetivo. Por exemplo, a tela inicial possui 3 cartões: Um com o título do programa, um com o formulário de login, e outro com o link para criar um novo usuário. A tela inicial está ilustrada abaixo. Perfil privado e público

Caso você deseje criar um sistema especialista com acesso privado, crie um perfil e faça o login com ele. Você também pode usar o perfil público, que já vêm preenchido por padrão no formulário de login, bastando clicar no botão entrar, porém este perfil, como diz o nome, é público e qualquer usuário poderá acessá-lo e modificar o conteúdo dos sistemas salvos nele.
